向量的期望和协方差

期望

  1. E[AXB+C]=AE[X]B+CE[AXB + C] = AE[X]B +CE[AXB+C]=AE[X]B+C
  2. E[xTAx]=tr(AΣ)+μTAμ,其中E[x]=μ,Var[x]=ΣE[\bm{x}^TA\bm{x}] = tr(A\Sigma) + \bm{\mu}^TA\bm{\mu}, \text{其中}E[\bm{x}] = \mu, Var[\bm{x}] = \SigmaE[xTAx]=tr(AΣ)+μTAμ,其中E[x]=μ,Var[x]=Σ

协方差

  1. Cov[x,y]=E[(x−α)(y−β)T], 其中α=E[x]Cov[\bm{x},\bm{y}] = E[(\bm{x} - \alpha)(\bm{y} - \beta)^T],\ \text{其中}\alpha = E[\bm{x}]Cov[x,y]=E[(x−α)(y−β)T], 其中α=E[x]

  4. Cov[ax+by,cu+dv]=acCov[x,u]+adCov[x,v]+bcCov[y,u]+bdCov[y+v]Cov[a\bm{x} + b\bm{y}, c\bm{u} + d\bm{v}] = acCov[\bm{x},\bm{u}] + adCov[\bm{x},\bm{v}] + bcCov[\bm{y},\bm{u}] + bdCov[\bm{y} + \bm{v}]Cov[ax+by,cu+dv]=acCov[x,u]+adCov[x,v]+bcCov[y,u]+bdCov[y+v]

幂等矩阵

命题：

  1. 幂等矩阵“诱导”出的矩阵仍是幂等矩阵。设 AAA 为 nnn 阶幂等矩阵，InI_nIn​ 为 nnn 阶单位矩阵，则
    (1) 若 BBB 与 AAA 相似，则 BBB 是幂等矩阵；
    (2) ATA^TAT、A∗A^*A∗、In−ATI_n-A^TIn​−AT 都是幂等矩阵；
    (3) 对任何正整数 kkk，AkA^kAk 也是幂等矩阵。
  2. 幂等矩阵的行列式。若 AAA 为幂等矩阵，则 det⁡(A)=0\det(A)=0det(A)=0 或 det⁡(A)=1\det(A)=1det(A)=1。
  3. 幂等矩阵的特征值。若 AAA 为幂等矩阵，则 AAA 的特征值只能为 000 或 111。
  4. 幂等矩阵的像空间“不变”。若 AAA 为幂等矩阵，则 Ax=xAx=xAx=x 的充要条件是 x∈im LAx\in \mathrm{im}\ L_Ax∈im LA​。

引理：

  1. 设 TTT 为有限维线性空间 VVV 上的线性算子，则下面两条性质等价：
    (1) ker⁡T+im T=V\ker T+\mathrm{im}\ T=VkerT+im T=V；
    (2) ker⁡T∩im T={0}\ker T\cap\mathrm{im}\ T=\{0\}kerT∩im T={0}。
  2. 幂等矩阵对应的直和分解。若 AAA 为幂等矩阵，则 ker⁡LA⊕im LA=V\ker L_A\oplus\mathrm{im}\ L_A=VkerLA​⊕im LA​=V。

定理：

  1. 幂等矩阵可相似对角化。设 nnn 阶幂等矩阵的秩为 r (1≤r≤n)r\ (1\leq r\leq n)r (1≤r≤n)，则 AAA 相似于对角阵 (IrO)\begin{pmatrix}I_r&\\&O\end{pmatrix}(Ir​​O​)

推论：

  1. 若幂等矩阵 AAA 的秩为 rrr，则 tr(A)=r\mathrm{tr}(A)=rtr(A)=r。
  2. 可逆的幂等矩阵为单位矩阵。
  3. 若 AAA 为 nnn 阶实对称幂等矩阵，则存在正交矩阵 QQQ，使得 Q−1AQ=(IrO)Q^{-1}AQ=\begin{pmatrix}I_r&\\&O\end{pmatrix}Q−1AQ=(Ir​​O​)。
